# 快捷键

从`v1.7.2`版本开始，在grid页面加入了几个操作快捷键，通过下面的方式开启

```php
$table->enableHotKeys();
```

下面是键和操作的对应关系

| 键    | 操作               |
| ----- | ------------------ |
| s     | 快捷搜索聚焦       |
| f     | 展开或者隐藏过滤器 |
| r     | 刷新页面           |
| c     | 进入创建页面       |
| left  | 跳转上一页         |
| right | 跳转下一页         |
